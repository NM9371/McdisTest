В докер компоуз добавлено и настроено все окружение : Kafka и PGSQL
Для автоматизации проверки настроено создание таблицы в БД.
После запуска контейнера запускается Kafka консюмер и с помощью паттерна Observer передает полученные события наблюдателю.
Наблюдатель в свою очередь при получении событий записывает статистику в БД добавляя строку, если строка с индексом (userID+eventType) уже есть то плюсуется Count.
Реализована фильтрация через переменные окружения EVENT_FILTER_START_DATE и EVENT_FILTER_END_DATE, 
при отсутствии одной она игнорируется, при отсутствии обоих фильтрация отключается. Изначально в docker-compose указан 2025 год.
