В докер компоуз добавлено и настроено все окружение : Kafka и PGSQL
Для автоматизации проверки настроено создание таблицы в БД, топика в Кафке и добавление тестовых сообщений в топик.
После запуска контейнера тестовые сообщения публикуются в топик, запускается консюмер и с помощью паттерна Observer передает полученные события наблюдателю.
Наблюдатель в свою очередь при получении событий записывает статистику в БД добавляя строку, если строка с индексом (userID+eventType) уже есть то плюсуется Count.
